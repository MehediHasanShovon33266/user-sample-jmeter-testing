# user-sample-jmeter-testing

How to generate report: 
    i) jmeter -n -t yourFile.jmx -l yourFile.csv
    ii)Delete the created csv file.
    iii)jmeter -n -t yourFile.jmx -l yourFile.csv -e -o Reports
